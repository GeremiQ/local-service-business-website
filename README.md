# Local Service Business Website Template

A responsive and performance-focused website template built for local service businesses.

## Live Demo

https://geremiq.github.io/local-service-business-website/

## Features

* Responsive multi-page website layout
* Service showcase with interactive carousel
* Dedicated service detail pages
* Contact page with quote request form
* Scroll-based reveal animations
* Image gallery with lightbox preview
* FAQ accordion sections
* Lazy-loaded and optimized images
* Deferred Google Maps embed
* SEO-friendly HTML structure
* GitHub Pages-ready relative asset paths

## Built With

* HTML
* CSS
* JavaScript
